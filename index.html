Q1. (Calculator) Write a program to make a simple calculator using switch statement that takes operator and two numbers from user and print the result after operation.

Ans. Input:- + 1 2 || Output :- 3
            const Calculator = (A, B, C) => {
                switch(A){
                  case "+" :
                    return (B+C)
                  case "-" :
                    return (B-C)
                  case "*" :
                    return (B*C)
                  case "/" :
                    return (B/C)
                }
            };

  • If X < Y, it means that X is smaller than Y.
  • If X > Y, it means that X is greater than Y.
  • If X = Y, it means that X is equal to Y.

Ans. Input :- 4567 ||  Output :- 4
            const findRelation = (x,y) => {
                if (x>y){
                  return (x + " is greater than " + y);
                }else if(x < y) {
                  return (x + " is smaller than " + y);
                }else{

                }
              };

Q5. (Find Grades) Your school has the following grading system based upon the marks(M) obtained by a student:
  • If M ≤ 10, the grade will be E.
  • If 11 ≥ M ≤ 20, the grade will be D.
  • If 21 ≥ M ≤ 30, the grade will be C.
  • If 31 ≥ M ≤ 40, the grade will be B.
  • If 41 ≥ M ≤ 50, the grade will be A.

Ans. Input :- 1  ||  Output :- E
            const findGrades = (a) => {
                if (a>=41 && a<=50){
                  return "A";
                }else if (a>=31 && a<=40){
                  return "B";
                }else if (a>=21 && a<=30){
                  return "C";
                }else if (a>=11 && a<=20){
                  return "D";
                }else if (a>=0 && a<=10){
                  return "E";
                }else{
                  return "error";
                }
              };

Q7. (Find the maximum out of three numbers.) Take three numbers and print the largest number among them if all of three are same print −1.

Ans. Input :- 2 5 4||  Output :- 5
            const Max_out_of_three = (A,B,C) => {
                if (A>B && A>C){
                  return A;
                }else if(B>A && B>C){
                  return B;
                }else if(C>A && C>B){
                  return C;
                }else{
                  return -1;
                }
              };

                if((x > y) && (x < z) || (x < y) && (x > z)){
                  return x;
                }else if((y > x) && (y < z) || (y < x) && (y > z)){
